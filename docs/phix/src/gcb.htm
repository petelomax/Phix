gcb

Definition: include theGUI.e

gbuf identifier [= gBuffer(...)]
-- or --
 gcb identifier [= gCanvas/gBuffer(...)]
Description: Type. An integer index into the private control tables of theGUI.
While a gbuf must have been created by a call to gBuffer,
          a gcb must be either from that or gCanvas.
Neither may be null.

Mainly intended for parameters to routines defined in theGUI itself, and/or to clarify the docs.
pwa/p2js: Supported/ignored as per gdx.
See Also: gdx
Technicalia Technically, gbuf is a strictly negative integer whereas gcb is either that or the positive gdx of a CANVAS entry, (where CANVAS is a private internal constant of theGUI.e) or the latter in fact HtmlElement in theGUI.js.

An application can freely use gbuf/gcb to define variables and clarify code, but it should not rely on implicit side-effects (other than sanity checks/helpful crashes) or explicitly use them for control flow purposes.
