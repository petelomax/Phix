cdCanvasTextAlignment

Syntax: include pGUI.e

integer prev_alignment = cdCanvasTextAlignment(cdCanvas hCdCanvas, integer alignment)
-- or --
integer prev_alignment = cdTextAlignment(integer alignment)
Description: Defines the vertical and horizontal alignment of a text as: CD_NORTH, CD_SOUTH, CD_EAST, CD_WEST, CD_NORTH_EAST, CD_NORTH_WEST, CD_SOUTH_EAST, CD_SOUTH_WEST, CD_CENTER, CD_BASE_LEFT, CD_BASE_CENTER, or CD_BASE_RIGHT.
Returns the previous value.
Default value: CD_BASE_LEFT.
Value CD_QUERY simply returns the current value.

cdTextAlignment(a) is equivalent to cdCanvasTextAlignment(cdActiveCanvas(),a).
