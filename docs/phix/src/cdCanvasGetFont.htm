cdCanvasGetFont

Definition: include pGUI.e

string fontname
integer fontstyle, fontsize

{fontname, fontstyle, fontsize} = cdCanvasGetFont(cdCanvas hCdCanvas)
-- or --
{fontname, fontstyle, fontsize} = wdCanvasGetFont(cdCanvas hCdCanvas)
Description: Returns the values of the current font, see cdCanvasFont for details.
Comments: In WC, the fontsize is returned in millimeters.
