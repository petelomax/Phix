current_dir

Definition: string s = current_dir()
Description: Return the name of the current working directory.
Comments: As shown below, get_proper_dir(".") may sometimes be a slightly better alternative to current_dir(), although it does not matter which is passed to join_path().
Example: 
s = current_dir()       -- s would be "C:\\Users\\Pete\\My Documents" if you were in that directory
s = get_proper_dir(".") -- s would be "C:\\Users\\Pete\\My Documents\\" if you were in that directory
See Also: dir, chdir, getenv, get_proper_dir, join_path
