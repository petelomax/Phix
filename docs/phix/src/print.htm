print

Definition: print(integer fn, object x, integer asCh=false, maxlen=-1)
Description: Print, to file or device fn, an object x with braces { , , , } to show the structure.

For more details on the optional asCh and maxlen parameters see sprint.
pwa/p2js: Supported, via sprint and puts, so the same applies here too.
There may be some '\n' differences on a console, but let me tell you straight without shame: I don't care.
[Of course, should you fiddle around with p2js.js/print() and send me an update, I'll likely obediently just quietly accept it.]
Example 1: 
print(1, "ABC")  -- output is:  "ABC"  (NB Euphoria displays {65, 66, 67})
Example 2: 
print(1, repeat({10,20}, 3))    -- output is: {{10,20},{10,20},{10,20}}
