gGetDialog

Definition: include theGUI.e

gdx res = gGetDialog(gdx id)
   -- or --
gdx res = gGetParent(gdx id)
   -- or --
gdx res = gGetBrother(gdx id, bool bPrev=false)
   -- or --
gdx res = gGetChild(gdx id, integer pos)
   -- or --
integer count = gGetChildCount(gdx id)
Description: Simple routines to help you navigate around a layout hierarcy.

id: Identifier of an interface element.
bPrev: return the next (default) or previous brother.
pos: position of the desired child starting at 1. (NB: IupGetChild() starts at 0)

gGetBrother(id,true) might, for instance, be used to get the preceding label of an input field.

Returns: an element/container/dialog or NULL if not found, except, obviously, for gGetChildCount(). The child/brother routines deliberately crash if passed a non-container, since that strongly indicates a logic error has almost certainly just happened. A gGetDialog(dlg) always returns itself, but always has a different parent. gGetBrother() does not wrap.
pwa/p2js: Supported, however you should refrain from printing a gdx or relying on any numerical ordering (in a browser).
