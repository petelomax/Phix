curl_multi_remove_handle

Definition: include builtins\libcurl.e

curl_multi_remove_handle(atom curlm, curl)
Description: Remove an easy handle from a multi session.

curlm: the multi_handle from which to remove an easy_handle. curl: the easy_hyandle to be removed from the multi_handle.

When the easy handle has been removed from a multi stack, it is again perfectly legal to invoke curl_easy_perform on this easy handle.

Removing an easy handle while being used is perfectly legal and will effectively halt the transfer in progress involving that easy handle.
All other easy handles and transfers will remain unaffected.

A fatal error occurs if the easy handle could not be successfully removed.
pwa/p2js: Not supported.
See Also: curl_multi_init, curl_multi_add_handle curl_easy_cleanup
