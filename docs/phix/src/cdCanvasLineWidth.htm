cdCanvasLineWidth

Definition: include pGUI.e

cdCanvasLineWidth(cdCanvas hCdCanvas, integer width)
Description: Configures the width of the current line (in pixels).
Comments: To retrieve the current value, use cdCanvasGetLineWidth(cdCanvas hCdCanvas).

If you find C code along the lines of old_width = cdCanvasLineWidth(ih,new_width);, it must be replaced with two calls.
Also replace current_width = cdCanvasLineWidth(ih,CD_QUERY) with current_width = cdCanvasGetLineWidth(ih).

In Phix, the procedure cdCanvasSetLineWidth is identical to cdCanvasLineWidth, and the width parameter is actually an atom, which gets rounded to the nearest int >= 1.
