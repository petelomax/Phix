Definition: integer status = task_status(integer task_id)
Description: Return the status of a task. Status values are 1 (active), 0 (suspended), and -1 (task does not exist)
Comments: A task might want to know the status of one or more other tasks when deciding whether to proceed with some processing. A -1 value could indicate a task that never existed, or a task that terminated naturally or was killed.
