print_json

Definition: include builtins\json.e
object res = print_json(object fn, object o, bool trim_res=false)
Description: Converts a JSON object to string representation.

fn: an open file handle (1 for the console), or "" to return a string.
o: a JSON object to print (see parse_json for details).
trim_res: ignored if fn is an integer, otherwise when true strips all whitespace and newlines from the returned result

Returns 0 if parameter o is not a valid JSON object, otherwise a string if fn was "", else fn unaltered.
pwa/p2js: Supported, but only when fn is "".
Notes: Attempts to print an invalid JSON object yield 0.
Note: it may get part way through before detecting an error.

Strings are automatically double-quote-escaped, but only when they need to be.
For instance `He said "No" firmly` is output as "He said \"No\" firmly" however input already in the latter form appears as-is.
Be advised that `\\"` will still emit invalid json, but properly pre-escaped strings shouldn't.
Also note that parse_json() makes no attempt to undo any such string mangling this may or may not have done.

The use of a named parameter when setting trim_res is recommended, to make the intent clear and the code easier to read.
See Also: parse_json
