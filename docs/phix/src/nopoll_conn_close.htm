nopoll_conn_close

Definition: include nopoll.e

nopoll_conn_close(atom conn)
Description: Close an opened noPollConn no matter its role (noPollRole).

conn: The connection to close.

There is available an alternative extended version that allows to send the status code and the error message: nopoll_conn_close_ext
See Also: nopoll_conn_close_ext
