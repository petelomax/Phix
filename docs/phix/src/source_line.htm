source_line

Definition: integer i = source_line()
Description: source_line() is a function built-in to the interpreter/compiler/transpiler.
It is replaced by the source code line, which can be useful for diagnostic purposes.
pwa/p2js: Supported, the line number refers to the original Phix source code rather than the transpiled JavaScript.
Comments: The call to source_line() costs nothing, except an unnamed symtab entry for the new literal constant.
It is optimized at compile-time into the appropriate integer, and only emits a load immediate literal.
Example: 
    ?source_line() -- displays 1, or whatever
    log_some_event("something",source_line())
Implementation: Optimised away as above - search for T_source_line in the compiler/transpiler source code for more details.
