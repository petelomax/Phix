proper

Definition: string res = proper(string s, sequence method="CAPITALISE")
Description: Perform selected upper/lowercase operations on a string.

s: the string to be modified
method: one of "LOWER","UPPER","CAPITALISE","SENTENCE",{<proper_nouns>},"INVERT".

Obviously LOWER/UPPER overlap functionally with lower()/upper(), except for being more dynamically specify-able.
SENTENCE is far from perfect, most obviously there is no list of proper nouns that should be capitalised, but one can be provided in method, which it makes a case insensitive copy of before carrying on by assuming SENTENCE.
  (A method of {} is taken to mean "re-use the one set up earlier without re-lowercasing it". See also technicalia.)
INVERT has (potential) use in theEditor, to correct something just typed in with the wrong caps lock setting, and it is probably not a terrible idea to automatically retry failed passwords using that option.

Albeit without the {<proper_nouns>} option, proper() is also integrated into the %s format of [s]printf(),
such that sprintf("%20/ls",s) returns the same as sprintf("%20s",proper(s,"LOWER")), with 'l' any of "lucsi".
pwa/p2js: Supported.
Example: 
constant s = "this is England. so there"
for m in {"LOWER","UPPER","CAPITALISE","SENTENCE","INVERT"} do
    printf(1,"%s (%s case)\n",{proper(s,m),proper(m)})
end for
printf(1,"%s (Proper nouns)\n",{proper(s,{"ENGLAND"})})
-- Output:
--  this is england. so there (Lower case)
--  THIS IS ENGLAND. SO THERE (Upper case)
--  This Is England. So There (Capitalise case)
--  This is england. So there (Sentence case)
--  THIS IS eNGLAND. SO THERE (Invert case)
--  This is England. So there (Proper nouns)
Implementation: See builtins\pcase.e (an autoinclude) for details of the actual implementation.
See Also: upper, lower
Technicalia Unlike the Euphoria version, this routine deliberately only handles strings, and not nested sub-sequences.

Sentences/phrases that start in double quotes preserve the existing leading-letter-case, for instance:

printf(1,"-- %s\n",{proper(`the phrase "unSupported BEhaviour" means...`,"SENTENCE")})
printf(1,"-- %s\n",{proper(`he said "ThAt sucks. bye." and hung up. smart.`,"SENTENCE")})
-- The phrase "unsupported behaviour" means...
-- He said "That sucks. Bye." and hung up. Smart.

which is as fancy as it gets - as noted above, don't expect miracles or "proper intelligence" from sentence case!
