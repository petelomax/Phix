ba_new

Syntax: bigatom ba = ba_new(object x)
Description: Create a bigatom instance value.
Comments: The parameter x can be integer, atom, or string. Obviously all the limitations for integer and atom parameters apply, whereas string parameters can be exact, to however many decimals you can type.

Apart from available memory, there are no known limits to precision, however we can make some observations about the likely limits of exponents. While (on a 32 bit system) the largest atom is around 1e308, so the largest exponent is 308, for a bigatom it is the exponent itself that has a limit of around 1e308. Errors are bound to creep in for any exponents over about 9007199254740992. Writing a bigbigatom library, which allows the use of bigatom exponents, is left as an exercise for the reader.
Example: 
include bigatom.e
bigatom ba = ba_new("0.123456789012345678901234567890123456789E+987654321")
See Also: ba_new, ba_scale
