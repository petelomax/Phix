close

Syntax: close(integer fn)
Description: Close a file or device and flush out any still-buffered characters.
Comments: Any still-open files are automatically closed when a program terminates (see technicalia).
See Also: open, flush
Technicalia Invoking close(-9) closes all open files. This is automatically performed at shutdown, and only intended for that use; applications should not make such a call explicitly, and no support is offered for any problems that doing so may (or may not) cause.
