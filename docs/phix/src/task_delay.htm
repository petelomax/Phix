task_delay

Syntax: task_delay(atom delaytime)
Description: Suspends a task for a short period, allowing other tasks to run in the meantime.

delaytime : an atom, the duration of the delay in seconds.
Comments: This procedure is similar to sleep(), but allows for other tasks to run by yielding on a regular basis.
Like sleep(), its argument need not be an integer.
See Also: sleep, task_yield, task_suspend
