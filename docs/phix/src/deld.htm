deld

Syntax: deld(object key, integer tid=1)
Description: Delete a {key,data} pairing from the dictionary.
Comments: If the specified key does not exist, no error occurs, and no changes are made.
Example: 
deld("key")
Technicalia Should it be highly likely that a key does not exist, preliminary calls to getd_index() may eliminate some re-balancing checks, and hence prove slightly faster.
