nopoll_msg_opcode

Definition: include nopoll.e

integer opcode = nopoll_msg_opcode(atom pMsg)
Description: Get the type of message that was received.

pMsg: The message that is being checked for its opcode

Returns: The opcode or -1 (NOPOLL_UNKNOWN_OP_CODE) in the case NULL reference is received.
See Also: nopoll_msg_opcode_desc
