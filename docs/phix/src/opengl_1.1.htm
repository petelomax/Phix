OpenGL 1.1

These routines are not supported by WebGL. (see the menu on the left for the full list)

While still (semi-)supported on desktop/Phix, many routines have been coloured illegal to indicate they cannot be handled by pwa/p2js, or perhaps more pertinently that Chrome/DuckDuckGo/Edge/Firefox/Lynx/Opera/Safari or whatever web browser you/others are using to handle the output it would have thrown in their general direction if it didn't already know not to bother trying. I did once hunt for a compatibility shim, and only found a single half-baked completely unusable mess, so can only assume that must all be on the far wrong side of "difficult". Likewise don't be expecting GLUT/GLU/FreeGLUT/GLFW/GLEW/SDL and the like to be browser or p2js compatible, or for that matter any support from me for those on desktop/Phix.
