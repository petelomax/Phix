drawShadedPolygonToDib

Definition: drawShadedPolygonToDib(a32Dib dib_dest, sequence coords_dest, sequence colors_dest)
Description: Draws a shaded polygon on bitmap dib_dest, with coordinates coords_dest {{x1, y1}, {x2, y2}, ...} (a polygon) and colors colors_dest {{b1, g1, r1}, {b2, g2, r2}, ...}.
Comments: Each color in colors_dest corresponds to a coordinate in coords_dest.
The polygon defined by coords_dest must be convex.
If you pass an empty sequence {} for coords_dest, then the bounding rectangle of the bitmap will be used.
