ba_round

Syntax: bigatom ba = ba_round(object N, atom precision=1, integer mode=0)
Description: Round to specified precision using the specified rounding mode.

N can be an atom, string, or bigatom.

A negative precision is the same as the positive reciprocal, eg -100 is the same as 0.01.

The rounding mode can be >0 for ceiling, <0 for floor, or 0 for nearest.

Comments: The original comments state -0.5 ==> -1 instead of -0.5 ==> 0, presumably when mode is 0.

It may be prudent to experiment with ba_round, ba_floor, and ba_trunc to figure out which one you actually want!
Example: 
include bigatom.e
bigatom ba = ba_round("-0.5")
