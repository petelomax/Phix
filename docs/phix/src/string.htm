string

Syntax: string identifier
-- or --
integer i = string(object x)
Description: This serves to define the string type.
You can also call it like an ordinary function to determine if an object is a string.
Comments: When invoked as a function, returns 1 if x is a string otherwise returns 0.

A type check error occurs if a variable declared as a string is assigned to a dword-sequence.
Example 1: 
string s
s = "hello"
Example 2: 
if string(x) then
    puts(1,x)
else
    -- x must be an atom or dword-sequence
    ? x
end if
