curl_slist_free_all

Definition: include builtins\libcurl.e

curl_slist_free_all(atom slist)
Description: Free an entire curl_slist list

curl_slist_free_all() removes all traces of a previously built curl_slist linked list.
Example:
Copy 
atom slist = curl_slist_append(NULL, "X-libcurl: coolness")
if slist==NULL then return -1 end if
curl_easy_setopt(some_handle, CURLOPT_HTTPHEADER, slist)
curl_easy_perform(some_handle)
curl_slist_free_all(slist)
See Also: curl_slist_append
